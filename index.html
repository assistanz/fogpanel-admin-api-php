    Simple library interface for FogPanel Admin APIs

    Project maintained by assistanz Hosted on GitHub Pages — Theme by mattgraham

    Install API Library

    The API Library provides a wrapper for all the APIs provided by the FogPanel Admin module.

    1. Download Composer:

      curl -s https://getcomposer.org/installer | php

    2. Add FogPanel Admin API as a dependency to your composer.json file:

      {
    "require": {
        "assistanz/fogpanel-admin-api": "dev-master"
    }
}

    3. Install Composer:

      php composer.phar install

    How to use?

    You should have API key and secret from FogPanel server. The server should be available online with active account to test.
